After a disaster, local communities and neighborhoods come together to support each other. Often times, it's frustrating and difficult to locate necessary supplies before federal aid arrives. SupplyCache allows residents to submit information via text message or web form, which then populates to a map for easy visualization. The use of clustered markers on the map help show the number of residents with supplies readily available while an added heat map layer shows areas affected by the disaster.

The name SupplyCache comes from the idea of Geocaching, in which people go find "hidden treasure" based on GPS coordinates and descriptions.

This is a project by Amy Lin, an alumnae from the Fall 2013 class of Hackbright Academy. The Github page for this project can be found here.
